# Julia Learning Journey

Welcome to your Julia learning workspace! This directory contains resources and examples to help you learn Julia programming.

## Getting Started

### Basic Commands
- `julia` - Start the Julia REPL (interactive shell)
- `julia script.jl` - Run a Julia script
- `julia --help` - Show help options

### Files in this directory
- `01_basics.jl` - Basic syntax and operations
- `02_functions.jl` - Functions and methods
- `03_data_structures.jl` - Arrays, dictionaries, and more
- `04_control_flow.jl` - Loops and conditionals
- `05_packages.jl` - Working with packages
- `examples/` - Practical examples

### Useful Resources
- [Official Julia Documentation](https://docs.julialang.org/)
- [Julia Academy](https://juliaacademy.com/)
